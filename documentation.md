# DRAWING PAD USING PYTHON

## 1. Abstract

The Drawing Pad is a Python-based graphical application developed using the Tkinter library. It allows users to draw freely on a digital canvas using mouse movements. The project demonstrates the implementation of graphical user interfaces, event handling, and canvas operations in Python.

## 2. Introduction

Drawing applications are widely used for creating sketches, diagrams, and digital artwork. The Drawing Pad project provides a simple platform where users can draw using a mouse. The application offers a clean canvas and a button to clear the drawing area.

## 3. Objectives

* To develop a drawing application using Python.
* To demonstrate GUI programming concepts.
* To implement mouse event handling.
* To create an interactive user experience.

## 5. Methodology

The application creates a graphical window using Tkinter. A canvas widget is used as the drawing area. Mouse movement events are captured, and small circles are continuously drawn on the canvas to simulate drawing. A clear button is provided to erase all drawings.

## 6. Algorithm

1. Start the application.
2. Create the main window.
3. Create a drawing canvas.
4. Detect mouse movement while the left button is pressed.
5. Draw on the canvas based on cursor position.
6. Provide a button to clear the canvas.
7. Continue until the application is closed.
8. End.
